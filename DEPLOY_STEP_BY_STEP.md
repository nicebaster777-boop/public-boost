# Деплой по шагам

## Шаг 1: Подключиться к серверу

```bash
ssh root@ваш-ip-адрес
```

**Или если у вас другой пользователь:**
```bash
ssh username@ваш-ip-адрес
```

**После подключения вы увидите приглашение командной строки сервера.**

---

**Напишите, когда подключитесь, перейдем к шагу 2.**

---

## Шаг 2: Обновить систему

```bash
sudo apt update && sudo apt upgrade -y
```

**Подождите, пока команда выполнится (может занять 2-5 минут).**

---

**Напишите "готово", когда команда выполнится.**
